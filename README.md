# Hospital Appointment and Record Management System (HARMS)

### Description
This project/program aims to help patients and doctors by providing an efficient system of appointments and medical records.

### Objectives
1. Provide a Function Appointment and Records system
2. Increase Hospital efficiency
3. Protect and filter users from confidential information
4. Provide and organize patient data
5. Provide an easy-to-use interface


### Features
User Log-in System
Patient ID System
Appointment Booking
Medical Records System

4. Run the program by clicking the green button on the top left corner <img width="32" height="24" alt="image" src="https://github.com/user-attachments/assets/c15c4a7a-8b7e-430f-8112-0e08d039fea5" />
5. You will be faced with the interface, you can either choose to Register, Login, or exit, enter 1-3 for their respective outcome .

6. Register yourself first; you’ll need to identify yourself if you’re a patient, doctor, or staff member.

7.  Then enter a username

8. Then a password

9. After that, you’ll be registered and will be brought to the main menu.

10. Then enter to Login
11. For the patient option
11.1. Enter your username and password

11.2. Enter the personal details required for the Medical record.

11.3. Then the system would give you 4 options: View records, Book appointments, View appointments, and Logout

	11.4 For View Record, enter 1.
 
	11.5 For Book Appointments, Enter 2, then enter your desired time for the appointment, make sure it’s in the format 9:00 AM

	11.6 For View Appointments, Enter 3.

	11.7 For Logout, Enter 4, You will be back at the Menu.

12. For the Doctor Option, you will need to register(refer to steps 6-8) a new account as a Doctor, or you can use a doctor account if you already have one. You will be given a Records & Appointment menu.

12.1 For View Patient Record, Enter 1. You will be asked for a patient ID. You can input P1 for now, but you can input a specific ID if you have one. 

12.2 For Edit Patient Record, Enter 2.  Same as 12.1, You will be asked for a patient ID, input P1. 

	12.3 Enter the Patient’s New Disease/Injury/Condition and etc.

	12.4 For View Appointments, Enter 3.

	12.5 For Logout, Enter 4.
13. For Staff, it will be the same as for doctors. 
14. In the Records and Appointments Menu, enter 3 to exit the system. Thank you For using our system!
